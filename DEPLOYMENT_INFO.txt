KEY CONCEPTS EXPLAINED
================================================================================

1. ATTENTION MECHANISM
   Formula: Attention(Q,K,V) = softmax(QK^T/√d_k)V
   - Interactive sliders show weight distributions
   - Real-time visualization of attention patterns
   - Understanding of softmax normalization

2. MULTI-HEAD ATTENTION
   Formula: MultiHead(Q,K,V) = Concat(head₁,...,headₕ)W^O
   - Adjustable number of heads (1-8)
   - See parallel processing benefits
   - Concatenation and projection visualization

3. POSITIONAL ENCODING
   PE(pos,2i) = sin(pos/10000^(2i/d_model))
   - Sinusoidal pattern visualization
   - Adjustable model dimensions
   - Understanding relative position encoding

4. FEED-FORWARD NETWORK
   FFN(x) = max(0, xW₁ + b₁)W₂ + b₂
   - Multiple activation functions
   - Dimension expansion visualization
   - Non-linearity injection

5. ENCODER-DECODER FLOW
   - Input tokens → Embedding → Positional Encoding
   - Encoder: Self-attention stacks
   - Decoder: Self-attention + Cross-attention
   - Output: Linear + Softmax

================================================================================
MATHEMATICAL FORMULAS INCLUDED
================================================================================

1. Scaled Dot-Product Attention
   Attention(Q, K, V) = softmax(QK^T/√d_k)V

2. Multi-Head Attention
   MultiHead(Q,K,V) = Concat(head₁,...,headₕ)W^O
   where head_i = Attention(QW_i^Q, KW_i^K, VW_i^V)

3. Feed Forward Network
   FFN(x) = max(0, xW₁ + b₁)W₂ + b₂

4. Positional Encoding
   PE(pos,2i) = sin(pos/10000^(2i/d_model))
   PE(pos,2i+1) = cos(pos/10000^(2i/d_model))

TROUBLESHOOTING

If website doesn't load:

     ps aux | grep -E "(8890|transformer)" 
  
  2. Try local URL first:
     http://localhost:8890/index.html

     cd /data/.openclaw/workspace/transformer-learning/

     lt --port 8890 --subdomain transformer-learning-guide &

  1. Check browser console for errors
  2. Try refreshing the page
  3. Clear browser cache

  2. LocalTunnel service may rate-limit
  3. Use local URL as fallback
